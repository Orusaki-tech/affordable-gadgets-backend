# Ad-hoc scripts

One-off debugging, testing, and verification scripts. Not part of the main app.

- **check_*** – API/upload/visibility checks
- **test_*** – Manual test scripts
- **verify_*** – Credential/deployment verification
- **profile_*.js** – API profiling (e.g. k6) scripts

Run from project root, e.g. `python scripts/adhoc/check_inventory_api.py`. Remove or archive when no longer needed.
